本文记录本人在日常工作中遇到的TypeScript的小问题

关于 extends 已有 interface 后需要忽略一些不需要参数的问题

假如你有一个这样的interface:

    interface A {
        name: string,
        age: number,
        job: string,
    }

你需要extends 这个 A 类型 到 B类型：

    interface B extends A {}

但你不需要其中的job，那么怎么办？这么办：

    interface B extends Omit<A, 'job'> {}
